## Important security note
This app uses a front-end login gate because it is meant to run for free on GitHub Pages.
That keeps casual visitors out, but it is **not true backend security**. Anyone with enough technical knowledge could inspect the files.
For real private security, you would need a backend or a private app platform.

## GitHub Pages
1. Create a new public repo.
2. Upload the four files at the repo root.
3. Go to Settings > Pages.
4. Set Source to Deploy from a branch.
5. Choose main and /(root).
6. Save.
7. Wait 1–2 minutes and open the live link.
